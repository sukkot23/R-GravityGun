## 플러그인 설명서
```
Player의 Vector값을 임의로 조정하여 플레이어를 조작할 수 있습니다
```

### 명령어
```
/velocity <player> : 해당 플레이어를 조정하는 중력건을 지급합니다.

 * 플레이어 이름의 일부분만 입력할 경우 비슷한 유저 모두에게 중력건에 영향을 미칩니다
 * 해당 플레이어만 조정할 경우 반드시 닉네임을 똑같이 입력해주세요
```
